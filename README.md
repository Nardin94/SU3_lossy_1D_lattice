# Project
Simulates the (SU(3) symmetric) quantum dynamics of a three-flavored lossy Fermi gas hopping on a 1D lattice, using quantum trajectories. The effective Hamiltonian can also be diagonalized and studied (e.g. to analyze the dark states)

## Building

Build the project 

mkdir build 
cd build 
cmake .. 
make 


and run it with ./SU3_QuantumTrajectories 

Simulation results are written to: ./output
